## Objectif
Créer un **dashboard comptable** à la Odoo sur la page d'accueil (`/` → `src/app/page.tsx`), qui remplace la redirection actuelle vers le plan comptable.

Le dashboard doit montrer :
1. **KPIs globaux** (carton en haut)
   - Résultat net (produits classe 7 − charges classe 6)
   - Total trésorerie (soldes débiteurs des comptes 5xxxxx)
   - Chiffre d'affaires (total crédits comptes 70xxxx)
   - Total charges (total débits comptes 6xxxxx)
   - Nombre de pièces non-annulées
   - Nombre de brouillons en attente

2. **Cartes par journal** (grille responsive)
   - Code et libellé du journal
   - Nombre de pièces (total / brouillons)
   - Solde total (débit − crédit des lignes)
   - Dernière date d'écriture
   - Couleur d'accent par type de journal (ACH → orange, VT → vert, CAI → bleu, BIMA → violet, OD → gris, PE → jaune, RAN → rouge)

3. **Lien direct** vers chaque page depuis les KPIs

## Architecture
- **Server action** : `src/server/dashboard.ts` — fonction `getDashboardStats(dossierId: string): DashboardStats`
- **Tests** : `src/server/dashboard.test.ts` — TDD, écrits AVANT l'implémentation
- **Page** : `src/app/page.tsx` — composant client avec le dashboard
- Le dossier par défaut s'appelle "Les Associés SA" (seed)
